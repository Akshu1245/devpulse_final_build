DevPulse is the unified platform for modern engineering teams to secure their APIs and monitor AI infrastructure spend without leaving their IDE. Built by K S Akshay at Rashi Technologies, Bangalore.

## Quick Install

Open VS Code, press `Ctrl+P`, and run:
```bash
ext install rashi-technologies.devpulse
```
